Description

This is a place to post and reference recipes for configuring the GlideinWMS system for certain types of jobs, tasks, or other use cases.

Recipe Description
Cloud Gives an example on how to configure your Factory and Frontend to submit to "Amazon EC2"-like clouds
Roll your own VM Gives examples on how to roll your own VM on Amazon EC2
Glideins in a Vacuum Instrictions to validate a worker node using "Glideins in a Vacuum" (From the Factory Operation pages).
Batch Gives an example on how to configure your Factory and Frontend to do direct batch submission via BOSCO
Token Auth Documents how to enable HTCondor Token Authentication between a Frontend and Factory Entry Points which support it
Multiple factories, single Frontend Gives an example on how to configure your Frontend to talk to multiple factories
