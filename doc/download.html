Current release distributions

The GlideinWMS software is available via OSG as RPM (the Frontend, the Factory and HTCondor with special configurations). The supported platforms are currently RHEL9, RHEL8 and RHEL7 (and derivates like Alma Linux, CentOS and SL).

GlideinWMS Frontend RPM

GlideinWMS Frontend RPM is available for download from the OSG RPM repository. The installation process will install and setup also the User Pool and User Schedd. Refer to OSG GlideinWMS Frontend installation guide for details.

GlideinWMS Factory RPM

GlideinWMS Factory RPM is available for download from the OSG RPM repository. The installation process will install and setup also the WMS Pool. Refer to OSG GlideinWMS Factory installation guide for details.

Downloading from the Code Repository

The code is maintained on GitHub. To checkout the version from git, run following commands (check the latest tag for each series)) 

 git clone https://github.com/glideinWMS/glideinwms.git
 # Stable Series (v3_10_x)
 git checkout v3_10_5
 # Development Series (v3_11_x)
 git checkout v3_11_0

GlideinWMS Cloud VMs

This activity is tracked separately under a different repository in GitHub (with an obsolete Redmine project, https://cdcvs.fnal.gov/redmine/projects/gwms-cloud-vms). See the cloud-vms repository for details. Glidein pilot rpms are available from the GlideinWMS yum repository. For details check GlideinWMS YUM Repository
