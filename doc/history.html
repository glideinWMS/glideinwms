Compatibility Statement

GlideinWMS strives to have Factory and Frontends compatible across versions. If there is an incompatibility:

  • Any incompatibility will be always clearly stated in the release notes
  • We will not require to have to upgrade Factory and Frontends at the same time. If this is unavoidable, a major version change should happen
  • If a change causing incompatibility is needed, we will prefer to require to upgrade the Factory before the Frontends and not the opposite (i.e. the newer Factory will be compatible with the older Frontends). If this happens, at least a minor version change should happen.
